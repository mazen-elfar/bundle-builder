# Bundle Builder

A production-ready Bundle Builder application built with **React**, **TypeScript**, **Vite**, and **Tailwind CSS**.

This project is a responsive, data-driven security system bundle builder that allows users to configure products, manage quantities, review their selections, and save their configuration locally.

---

## ✨ Features

- Multi-step accordion bundle builder
- Dynamic product rendering from JSON
- Variant (color) selection
- Quantity management
- Live order summary
- Automatic price calculation
- Responsive design
- LocalStorage persistence

---

## 🛠️ Tech Stack

- React 19
- TypeScript
- Vite
- Tailwind CSS v4
- React Compiler

---

## 📦 Installation

Clone the repository:

```bash
git clone https://github.com/mazen-elfar/bundle-builder.git
```

Install dependencies:

```bash
npm install
```

Run the development server:

```bash
npm run dev
```

Build for production:

```bash
npm run build
```
